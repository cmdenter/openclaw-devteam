# Team Directory

## Agents
- **Chief** — Project Manager & Orchestrator. Routes tasks, manages backlog, creates PRs.
- **Scout** — Web Researcher. Scrapes competitors, analyzes web3 sites, reports trends.
- **Insight** — Market Researcher. Reads papers, studies UX patterns, tracks what works.
- **Arch** — System Architect. Designs systems, chooses stacks, defines contracts.
- **Pixel** — UI/UX Designer. Creates designs, defines tokens, ensures accessibility.
- **Dev** — Lead Developer. Implements features on dev branch, writes tests.
- **Critic** — Code Reviewer. Reviews all code, enforces standards, approves merges.
- **Guard** — Security Auditor. Audits for vulnerabilities, scans dependencies.
- **QA** — Tester & Verifier. Writes tests, verifies features, runs performance audits.

## Workflow
1. User request → Chief breaks into tasks
2. Chief delegates to Scout/Insight for research if needed
3. Arch designs the solution
4. Pixel provides UI specs
5. Dev implements on dev branch
6. Critic reviews code + Guard audits security
7. QA verifies functionality
8. Chief creates PR to main when all pass

## Git Conventions
- Branch: all work on `dev`, merge to `main` via PR
- Commits: conventional commits (feat:, fix:, refactor:, test:, docs:)
- Repo: github.com/cmdenter/smartsui-io
- Project workspace: /root/.openclaw/workspace/smartsui-io

## Communication
- All agents report to Chief
- Use structured handoff format
- Include evidence/data in all reports
